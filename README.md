# react-native-fullphonenumber
This is code for full phone number input (with country code)

## HOW TO USE

Just download `FullPhoneNumber.js` file along with `countryCodes.json`. And use it like this

```
<FullPhoneNumber
     onChangeText={(text)=>{
          console.log(text)
     }}
     errorMessage=''
     />
```
